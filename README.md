Personal Finance Tracker:

A full-stack MERN application to track daily expenses, manage budgets, and visualize financial insights with an intuitive UI.

 Features:
 User Authentication (JWT-based signup & login)
 Add, Edit, Delete Transactions
 Categorized Expenses (Food, Travel, Shopping, Bills, etc.)
 Monthly Expense Tracking
 Dashboard Insights
 Responsive UI built with React
 MongoDB Atlas for data storage
 REST API using Node.js & Express

 Tech Stack:
       Frontend: React.js
                 Axios
                 Context API / useState / useEffect
                 CSS
       Backend: Node.js
                Express.js
                JWT Authentication
                MongoDB + Mongoose
